## 📙 Task 3: Bayes’ Theorem in Real Life

### Overview
Explanation of Bayes’ Theorem and its applications in real-world scenarios.

### Formula
P(A|B) = [P(B|A) × P(A)] / P(B)

### Applications
- Medical Diagnosis  
- Spam Filtering  
- AI & Machine Learning  
- Weather Prediction  

### Learning Outcome
- Understanding probability in decision-making
- Application of statistics in AI
